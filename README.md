## 📌 Project Overview

This project analyzes Netflix Movies and TV Shows using Python and data visualization techniques. The dataset contains over 8,800 Netflix titles, and the analysis uncovers trends in content types, genres, ratings, countries, and content growth over time.

---

## 🚀 Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📊 Analysis Performed

✅ Movies vs TV Shows

✅ Top 10 Countries Producing Netflix Content

✅ Netflix Ratings Distribution

✅ Most Popular Genres

✅ Content Added Over the Years

✅ Top Directors on Netflix

## 📈 Key Insights

* Movies dominate Netflix content.
* The United States contributes the largest number of titles.
* Netflix content grew significantly after 2015.
* Drama and International Movies are among the most popular genres.
* Certain directors have contributed multiple titles to the platform.
